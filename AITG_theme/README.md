# Advance-IT
 # Project Title: AITG Company Website

## Overview

This project is a modern and responsive website for AITG, a company specializing in software development, IT consulting, and digital transformation. The website is designed to be visually appealing and user-friendly, providing visitors with comprehensive information about the company's services, team, and contact details. It features a clean and professional design with a focus on user experience, incorporating animations and interactive elements to engage visitors.

## Features

- **Homepage (`index.html`):** A welcoming introduction to the company, highlighting key services and strengths.
- **About Page (`about.html`):** Detailed information about the company's mission, vision, and the team behind it.
- **Contact Page (`contact.html`):** A comprehensive contact form with options for visitors to specify their interests, along with contact information and a map.
- **News/Blog Page (`news.html`):** A blog section with articles on industry trends, company news, and insights.
- **Responsive Design:** The website is fully responsive and optimized for a seamless experience across desktops, tablets, and mobile devices.
- **Animations:** Subtle animations and transitions are used to enhance the user experience and guide visitors through the content.
- **Interactive Elements:** The site includes interactive elements such as a newsletter subscription form and a dynamic contact form.

## Technologies Used

- **HTML5:** The core markup language for creating the web pages.
- **CSS3:** Used for styling the website, with features like Flexbox, Grid, and custom animations.
- **Bootstrap 5:** A popular CSS framework for building responsive and mobile-first websites.
- **JavaScript:** Powers the interactive features of the website, including the news blog and contact form.
- **AOS (Animate On Scroll) Library:** Used for creating scroll-triggered animations.
- **Owl Carousel:** A touch-enabled jQuery plugin that lets you create a beautiful responsive carousel slider.
- **jQuery:** A fast, small, and feature-rich JavaScript library that simplifies HTML document traversing, event handling, and animating.

The project is organized into the following directories and files:
